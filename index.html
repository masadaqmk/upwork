See budget, client rating, total spend, hire rate, proposal count, and skills — all extracted automatically from the job page.

AI Match Scoring

Get a percentage match score comparing the job's requirements to your profile. Know immediately if it's worth your time to bid.

Fake Job Detector

AI analyzes red flags like unrealistic pay, vague descriptions, new accounts, and suspicious requests to protect you from scams.

How it works
Set up in under 2 minutes
No accounts, no subscriptions. Just install, add your API key, and start winning jobs.
1

Install the Extension

Add to Chrome from the Web Store. It's completely free with no sign-up required.

2

Add Your API Key

Go to Settings and enter your OpenAI API key. It's stored only on your device — never on our servers.

3

Open a Job Listing

Browse Upwork as usual. Click the extension icon on any job page to start the analysis.

4

Win the Job

Review your AI match score, run the fake job check, and generate a winning proposal — all in one click.
